# D&V Materiales

Catálogo público de D&V Materiales, Totoras, Santa Fe.

La publicación se realiza automáticamente con GitHub Pages al actualizar la rama `main`.

## Dominio propio (pendiente)

La web continúa publicada en GitHub Pages. Cuando se confirme la compra de
`dyvmateriales.com.ar`, se deberá:

1. agregar el archivo `CNAME` con el dominio elegido;
2. configurar DNS y HTTPS en GitHub Pages;
3. elegir una sola versión canónica (`www` o sin `www`) y redirigir la otra;
4. actualizar canonical, Open Graph, JSON-LD, `robots.txt` y `sitemap.xml`;
5. comprobar que la dirección de GitHub no genere contenido duplicado.

No se incluye un archivo `CNAME` todavía para no modificar la URL pública actual.
